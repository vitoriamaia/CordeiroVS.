# Skeleton ( Wordpress ) #

É um modelo de start para aplicações wordpress em ambiente de desenvolvimento que utiliza metodologias ágeis, onde conta com as melhores tecnologias do mercado para ajudar no trabalho do desenvolvedor. 

### Versão ###
   2.0
